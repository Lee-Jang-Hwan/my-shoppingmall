# E2E 테스트 체크리스트

이 문서는 배포 후 수동 테스트를 위한 체크리스트입니다. 각 항목을 테스트하고 결과를 기록하세요.

## 사용 방법

1. 각 섹션의 항목을 순서대로 테스트
2. 체크박스를 사용하여 완료 여부 표시: `[x]` 완료, `[ ]` 미완료
3. 발견된 이슈는 "발견된 이슈" 섹션에 기록
4. 모든 테스트 완료 후 결과를 정리
